# Famous Meme Research Cache

This registry points to canonical or owner-posted cat meme media without
redistributing the unlicensed mother files through Git.

Run:

```bash
npm run fetch:famous-memes
```

The script writes into `.cache/miao-famous-memes/`, which is ignored by Git.
Every entry is `research-only` until explicit commercial adaptation permission
is recorded. Generated calibrations are visual research, not production assets.
