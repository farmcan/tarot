# MiaoTarot Meme Archetype Map

Date: 2026-07-04

This document maps each Major Arcana card to the most useful internet-cat meme archetype before image-to-image production starts.

The goal is to decide what the raw meme base should be, not to commit famous meme images into the repo. Many canonical meme images are research-only unless rights are verified or the pose is recreated with owned/licensed photos.

## Decision Rule

For each card, pick the meme by behavior first:

```text
Tarot function -> Miao emotional state -> recognizable cat meme behavior -> production-safe base
```

Do not pick a meme only because it is famous. It must survive a mostly-preserved stylization pass.

Preservation modes:

| Mode | Meaning |
| --- | --- |
| Light wash | Preserve most of the original composition, pose, expression, and object relationship. Use only with owned/licensed/approved bases. |
| Tarot wash | Preserve pose/expression strongly, but allow background and props to change for Tarot symbols. |
| Research-only | Use the famous meme as archetype/vibe only; recreate with owned/licensed source. |
| Custom needed | No strong canonical meme fit; shoot, commission, or find a safer production image. |

## Priority Recommendations

Use these as the first three image-to-image tests:

| Card | Best archetype | Why first |
| --- | --- | --- |
| The Tower | Cat knocking object off table | Highest confidence; original composition should be mostly preserved. |
| The Moon | Huh/shocked wide-eyed cat | Tests face preservation and mood styling. |
| The Fool | Zoomies/running cat | Tests motion preservation and Tarot symbol insertion. |

Then solve the five weak/mid base replacements:

| Card | Best archetype | Issue to solve |
| --- | --- | --- |
| The Hierophant | Ceiling Cat / ritual authority | Current `ZEN` is not meme-specific enough. |
| Wheel of Fortune | Maxwell / spinning cat | Current `WOBBLE` lacks a canonical rotating meme anchor. |
| The Chariot | OIIA spinning cat | Current base is too small; needs stronger motion source. |
| The Hanged Man | Upside-down cat / Huh Cat | Needs actual inversion, not just head tilt. |
| High Priestess | Staring Cat / Gusic | Needs a stronger iconic stare. |

## Prompt Implication

For cards marked Light wash, use high reference strength:

```text
Preserve the cat pose, facial expression, camera angle, and core object relationship.
Apply MiaoTarot styling and add only subtle Tarot symbols.
Do not replace the meme scene with a fantasy Tarot scene.
```

For cards marked Research-only, do not feed the famous meme image into production unless rights are cleared. Use the archetype to brief a recreated or licensed base.
